===============
 Setting up venv
===============

| mkdir my_project
| cd my_project
| python -m venv venv
| source venv/bin/activate

Save requirements.txt
pip freeze > requirements.txt

Install from requirements.txt

pip install -r requirements.txt

| deactivate venv
| deactivate
| requirements.txt...
